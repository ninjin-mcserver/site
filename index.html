How To Join

参加までの流れ

  1. 01

    Minecraft Java Edition のユーザー名を運営へ連絡します。

  2. 02

    ホワイトリスト登録後、マルチプレイから接続先を追加します。

  3. 03

    初回ログイン後、スポーン周辺の案内とルールを確認します。

Invitation

参加したい人へ

参加希望の人は、Minecraft Java Edition のユーザー名を運営へ送ってください。ホワイトリストに追加されたあと、接続情報のアドレスから参加できます。

Rules

参加ルール

建築と土地

人の建築物や準備中の土地は、本人に確認してから触ります。共有施設は看板やメモを残して使いやすくします。

アイテム

チェスト、畑、装置の中身は持ち主の許可を取ります。共有倉庫は補充できる範囲で使います。

探索

遠出や大きな探索の前には座標を控えます。危険な場所を見つけたら、入口に目印を置きます。

トラブル

事故や破損に気づいたら、隠さず運営へ連絡します。巻き戻しが必要な場合はバックアップから判断します。
